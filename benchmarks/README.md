## Benchmarks

Download the benchmark archive from `https://github.com/muhd-umer/ltr-lowrank-sdp/releases/download/files/dat-s_files.zip` via the link or using `wget` as:
```bash
wget https://github.com/muhd-umer/ltr-lowrank-sdp/releases/download/files/dat-s_files.zip
```
and unzip it in the `benchmarks/` directory, like so:
```bash
unzip dat-s_files.zip -d benchmarks/
```
